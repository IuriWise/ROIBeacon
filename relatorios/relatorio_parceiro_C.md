**Relatório Executivo: Análise do Teste A/B de Cashback**

**Introdução:**
O objetivo deste relatório é analisar os resultados do teste A/B realizado com dois grupos de usuários com diferentes estruturas de cashback. Com base nos dados fornecidos, identificaremos qual variante de cashback deve ser escalada para 100% do tráfego.

**Análise dos Dados:**

* **Grupo 1:**
 + Compradores: 4.549
 + Comissão: R$ 121.693,00
 + Cashback: R$ 86.924,00
 + Vendas Totais: R$ 1.738.460,00
* **Grupo 2:**
 + Compradores: 4.522
 + Comissão: R$ 117.967,00
 + Cashback: R$ 117.967,00
 + Vendas Totais: R$ 1.685.235,00

**Cálculos:**

* **Margem de Comissão Líquida (MCL) do Grupo 1:** (Comissão - Cashback) / Vendas Totais = (121.693 - 86.924) / 1.738.460 ≈ 2,0%
* **Margem de Comissão Líquida (MCL) do Grupo 2:** (Comissão - Cashback) / Vendas Totais = (117.967 - 117.967) / 1.685.235 = 0,0%
* **ROI (Retorno sobre o Investimento) do Grupo 1:** Comissão / Cashback = 121.693 / 86.924 ≈ 1,40
* **ROI (Retorno sobre o Investimento) do Grupo 2:** Comissão / Cashback = 117.967 / 117.967 = 1,00

**Conclusão:**
Com base nos dados financeiros reais, o Grupo 1 apresenta uma margem de comissão líquida de 2,0%, enquanto o Grupo 2 apresenta uma margem de comissão líquida de 0,0%. Além disso, o ROI do Grupo 1 é de 1,40, enquanto o ROI do Grupo 2 é de 1,00.

**Recomendação:**
Considerando os resultados, **recomendamos escalar a variante do Grupo 1 para 100% do tráfego**. Essa variante apresenta uma margem de comissão líquida positiva e um ROI mais alto, indicando um melhor desempenho financeiro. A variante do Grupo 1 parece ser mais eficaz em gerar receita e manter uma margem de comissão líquida saudável.
